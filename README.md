# 🌙 Eid Mubarak Animated Website

✨ A fully animated Eid Mubarak celebration website built using **HTML, CSS & JavaScript**.

This project features beautiful animations, interactive effects, and a festive Eid theme.

---

## 🎆 Features

- 🌙 Animated Eid Mubarak typing effect
- 🎇 Canvas Fireworks Animation
- 🎉 Celebration Button with burst effects
- 🌌 Gradient Night Sky Background
- 💫 Smooth UI and responsive layout
